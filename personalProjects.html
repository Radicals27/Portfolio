Personal Projects

Salvage (Working title)

Salvage is a WIP, and is a single player first person scifi exploration game. It draws inspiration from extraction games as well as survival games like The Long Dark and Alien Isolation.



Alien Standup

The first game I released to the public (webGL/itch.io). A cross between mad-libs and Mastermind, try to learn the Alien language while cracking some great jokes on an alien planet.



Danny's Defense

A multiplayer tower defense game where you and your friends have to defend the fridge from attacking garlic. This was a Secret Santa project I made for a colleague - Danny, who has an allergy to garlic. Each weapon is based on an interest of his; guitar, Gwent, Bundaberg ginger beer, coding, ping pong and tea. For this project I learned how to do network code as well as Blender; I modelled, rigged and textured all the weapons and characters.



Pete Simulator

Another Secret Santa project, this time for my boss, Pete. It simulates the stress of managing a company whilst things go wrong which need your attention. All the while having to track down grant money.



Planetlike (Rails-based dating website)

A dating website with user authentication, admin/user roles, image upload, multi-field search feature and messaging system. (Sign up does not require verification, so anyone can sign up and try, use a fake email if you prefer.)



Game of Death (Terminal App, Ruby)

A Ruby-based text-based-adventure with multiple user choices and combat!



Space Invaders (Python)



A take on the old classic, with randomly-placed enemies, written in Python.
